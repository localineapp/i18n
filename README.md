## 🌐 Overview

This repository contains all **internationalization (i18n) language files** used across the Localine ecosystem.

Each file represents a locale and provides the full set of translation keys required by the Localine platform and its integrations.

---

## 📁 Structure
- `en_US.json` - English (United States) translations (default locale)
- `de_DE.json` - German (Germany) translations

Each file contains a JSON object with translation keys and values.

Example:

```json
{
  "message.welcome": "Welcome to Localine!",
  "message.goodbye": "Goodbye and see you soon!"
}
```

## 🤝 Contributing

Contributions are welcome — especially new translations or improvements to existing ones.

1. [Fork](https://github.com/localineapp/web-app/i18n) the repository
2. Clone your fork (`git clone https://github.com/your-username/i18n.git`)
3. Create a new branch for your changes (`git checkout -b feat/add-french-translations`)
4. Add or update translations in the appropriate JSON file
5. Commit your changes (`git commit -m 'feat: add French translations'`)
6. Push to the branch (`git push origin feat/add-french-translations`)
7. Open a Pull Request
